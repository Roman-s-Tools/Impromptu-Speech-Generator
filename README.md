# Impromptu Speech Generator

A lightweight web app for practicing impromptu speaking.

## Features
- Random prompt generation for speech practice
- Clean, distraction-free interface
- Works entirely in the browser (no backend required)

## Project Structure
- `index.html` – app markup
- `style.css` – visual styling
- `script.js` – prompt logic and interactions

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in any modern browser.
3. Click the prompt button to generate a topic and begin speaking.

## Use Cases
- Speech and debate warmups
- Interview preparation
- Classroom public-speaking exercises
